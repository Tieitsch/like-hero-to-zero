# Like Hero To Zero

Prototype web application for the sustainability project **"Like Hero To Zero"**.

## Project Goal
The application visualizes publicly available CO2 emission data per country and allows registered scientists to add or correct emission data.

## Core Features (MUST)
- Public view: Display the latest available CO2 emissions for a selected country.
- Backend view: Authenticated scientists can add new emission data.

## Technology Stack
- Spring Boot
- Spring MVC
- Spring Data JPA (Hibernate)
- Thymeleaf
- MySQL (or H2 for development)
